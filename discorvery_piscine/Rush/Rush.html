Projects -



  • Push_Swap


    This project introduced me to optimized data sorting, the aim is to take a stack as an input, and using only rotate commands, push commands and one other stack, organize it in the most efficient way possible. I really enjoyed this assignment as it I love to take code I’ve written and tinker with it in ways to make it better. I’m particularly proud of this assignment as I made a point of not using any external resources or guides when making this and not settling until it got the maximum scores 42 gives for this assignment (100numbers < 700moves, 500numbers < 5500moves) and therefore I think you would be hard pressed to find a similar sorting code from other students.

  • Pipex


    This assignment is to introduce students to pipes and redirection, the aim is to create a program that imitates the shell pipe ‘|’ command. I found the concept of this assignment particularly confusing as prior to this I had very little experience with redirection, however with time and research I managed to get a good grasp on how pipes work and how piping outputs into other programs can be useful.

  • Fractol


    This was a pretty fun assignment to do, the goal was to create a program that displays a fractal on a window that can be adjusted and changed based on key and mouse inputs. This was fun to learn as it was my first introduction into using windows and using hooks to manipulate them and the images on the screen. I really enjoyed learning about how colors and bit manipulation work within programs and had a lot of fun coming up with creative ways to change the colors.

  • This Website


    This Website is actually the last project I’ve worked on, I decided to do this after completing the Web-Dev Discovery piscine and am particularly proud of it as it goes beyond the scope of what we learnt in that week and shows how much I can learn in just 5 days of study. I built this without the use of any extra libraries or website builders and included elements of HTML, CSS, JavaScript and PHP, all of which I had no experience with less than a week ago. I have found working on this website to be lots of fun, I really enjoyed learning how different parts of a website interact with each other and ways to tinker with the code to make it smoother or work better and would love to work in Web Development if given the chance.
